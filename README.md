# Festival de Música - Techno & EDM 🎧

Este es un proyecto de práctica profesional para un sitio web de un festival de música. El objetivo principal es implementar un flujo de trabajo moderno utilizando **Gulp** para la automatización de tareas y **Sass** para estilos avanzados.

## 🚀 Tecnologías Utilizadas

* **HTML5** - Estructura semántica.
* **Sass (SCSS)** - Preprocesador de CSS con arquitectura de archivos (Mixins, Variables, BEM).
* **JavaScript (Vanilla)** - Funcionalidades de galería, scroll suave y navegación fija.
* **Gulp** - Automatización de tareas:
    * Compilación de Sass a CSS.
    * Generación de Source Maps para depuración.
    * Optimización de imágenes (WebP, Avif, Minificación).
    * Monitoreo de cambios en tiempo real (Watch).

## 📁 Estructura del Proyecto

El proyecto se divide en dos directorios principales:

* `src/`: Contiene el código fuente (archivos Sass, JS moderno e imágenes originales).
* `build/`: Carpeta generada automáticamente por Gulp que contiene los archivos optimizados listos para producción.

## 🛠️ Instalación y Configuración

Para replicar este proyecto localmente, asegúrate de tener instalado [Node.js](https://nodejs.org/).
